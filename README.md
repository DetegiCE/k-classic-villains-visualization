# 고전소설 속 악인 33인 외모 묘사 분석

KU Digital Humanities And Classical Korean Literature

## 프로젝트 소개

한국 고전 소설에 등장하는 33명의 악인(혹은 주요 인물)들의 외모 묘사를 분석하고 시각화한 인터랙티브 웹사이트입니다.

## 주요 기능

### 메인 페이지
- **플로팅 뷰**: 인물 카드들이 부유하는 애니메이션 효과
- **리스트 뷰**: 그리드 레이아웃으로 모든 인물을 정리하여 표시
- 뷰 전환 버튼으로 두 가지 보기 방식 선택 가능
- 통계 페이지로 이동하는 버튼

### 상세 모달
각 인물 카드를 클릭하면 다음 정보를 포함한 모달이 표시됩니다:
- 자동 슬라이드되는 이미지 (2장)
- 인물 이름 및 등장 작품
- 외모 묘사 및 출처
- 추출된 명사/형용사 키워드
- 감성 분석 결과 (파이 차트)
- 긍정/부정/중립 분류
- 감성이 비슷한 악인 3명 추천 (클릭 시 해당 인물로 이동)

### 통계 페이지
- 명사 워드클라우드 및 빈도 막대그래프
- 형용사 워드클라우드 및 빈도 막대그래프
- 빈도에 따른 글자 크기 및 색상 변화

## 데이터 구조

각 악인 데이터는 다음 정보를 포함합니다:
- 등장 작품 제목
- 인물 이름
- 외모 묘사 및 출처
- 추출된 명사/형용사
- 감성분석 결과 (혐오/분노/공포/놀람/행복/슬픔/중립 퍼센트)
- 긍정/부정/중립 분류
- 이미지 2장

## 실행 방법

### 로컬 서버 실행
```bash
# Python 3 사용
python3 -m http.server 8000

# 또는 npm 스크립트 사용
npm start
```

브라우저에서 `http://localhost:8000` 접속

## 기술 스택

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Chart.js (데이터 시각화)
